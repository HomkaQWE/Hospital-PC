# 🏥 Medical Center Landing Page

## 📖 Описание
Адаптивный лендинг для медицинского центра. Проект демонстрирует навыки вёрстки многостраничного макета с блочной структурой, использование графических элементов и организацию файлов по папкам.

## 🛠️ Технологии
- **HTML5** — семантическая вёрстка
- **CSS3** — каскадные таблицы стилей
- **Flexbox / Grid** — построение сеток
- **БЭМ-подход** (начальный уровень) — организация классов
- **Git** — контроль версий
